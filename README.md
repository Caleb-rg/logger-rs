# logger-rs

## Install Rust
```bash
$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Run (Release Mode)
```bash
$ cargo run --release
```

or shorthand:
```bash
$ cargo r -r
```
